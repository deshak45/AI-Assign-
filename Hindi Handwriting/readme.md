# Hindi Handwriting Recognition
The above code classify different alphabets of hindi language.
# Description
This code recognizes hindi handwriting.
# Techniques
I used convolutional neural networks and I use Tensorflow as the framework and Keras API for high abstraction level.
# Code Requirements
You can install Conda for python which resolves all the dependencies.
